# SymptomSense | AI Health Assessment

SymptomSense is a modern, AI-powered health assessment application built with React and Vite. It allows users to input their symptoms and receive instant analysis, potential conditions, and urgency levels.

## Features

-   **Smart Analysis:** Instantly maps symptoms to potential conditions using a local database.
-   **Private & Secure:** All data is processed and stored locally on your device.
-   **Detailed Insights:** Get clear recommendations, urgency levels (Low, Moderate, High), and match confidence.
-   **History Tracking:** Save your assessments and track your health history over time.
-   **Responsive Design:** Beautiful, glassmorphism-inspired UI that works on all devices.

## Tech Stack

-   **Frontend:** React 18
-   **Build Tool:** Vite
-   **Styling:** CSS3 (Variables, Flexbox, Grid)

## Getting Started

### Prerequisites

-   Node.js (v14 or higher)
-   npm (v6 or higher)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/induririsheendra-cmd/SymptomSense.git
    cd SymptomSense
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser and navigate to `http://localhost:5173`.

## Deployment

This project is optimized for deployment on [Vercel](https://vercel.com).

1.  Push your code to a GitHub repository.
2.  Import the project into Vercel.
3.  Vercel will automatically detect the Vite configuration.
4.  Click **Deploy**.
